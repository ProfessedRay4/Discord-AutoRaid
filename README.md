# auto raid-discord

### Installation:
1) Download zip (or git clone)
2) extract zip
3) ``pip install -r requirements.txt``
4) goto [https://discord.com/developers/](https://discord.com/developers/)
5) edit last line (31)
6) run Main.py ``python main.py``
7) enjoy!
